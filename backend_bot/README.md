## 🛠 О проекте
Это профессиональный FAQ-ассистент с архитектурой **Retrieval-Augmented Generation (RAG)**. Система идеально подходит для компаний со сложными прайс-листами, строгими регламентами или большим ассортиментом (аптечные сети, клиники, дистрибьюторы медоборудования).

### Ключевые возможности:
1. **Интеллектуальный поиск (Semantic Search):** Бот понимает смысл вопроса, даже если клиент использует синонимы или делает опечатки.
2. **Гибкая база знаний:** Поддержка форматов `.txt` и `.json` позволяет быстро обновлять данные о тарифах, акциях и условиях доставки.
3. **Безопасность данных:** Система использует изолированный векторный индекс FAISS. Данные передаются только в контексте конкретного запроса.
4. **Готовый UI-виджет:** В комплекте идет современный чат-интерфейс, который можно интегрировать на любой сайт за 5 минут.

## 🏗 Структура системы

* `backend/` — "Мозг" системы на FastAPI: обработка запросов, поиск в базе знаний и связка с OpenAI.
* `data/` — Корпоративная база знаний (информация о компании, прайсы, FAQ).
* `frontend/` — Клиентская часть: адаптивный виджет чата для сайта.
* `build_index.py` — Инженерный скрипт для оцифровки документов и создания векторного хранилища.

## 🚀 Техническое развертывание

### 1. Подготовка среды
Создайте файл `.env` в корневом каталоге и укажите ваш ключ доступа:
```env
OPENAI_API_KEY=sk-your-key-here
```

### 2. Установка

```bash
# Создание виртуального окружения
python -m venv venv

# Активация окружения (Windows)
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt
```
### 3. Загрузка базы знаний

Для того чтобы бот "выучил" ваши документы, запустите индексацию:

```bash
python -m backend.build_index
```
### 4. Запуск сервиса

```bash
uvicorn backend.app:app --reload
```
API будет доступно по адресу: <http://localhost:8000/docs>
